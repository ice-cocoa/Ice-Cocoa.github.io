介绍CSS布局

在CSS中实现页面布局的主要方法是设定display属性的值。正常流中所有内容都有一个display值：

  • 表现为块级设置 display:block; 如<li>的display默认为block
  • 表现为内联设置为 display:inline;
  • 不会换行的块展示 display:inline-block;
  • 弹性盒子 display:flex;
  • grid布局 display:grid;

此外还有几种微调元素的操作，如浮动和定位，它们对设计特殊元素很有帮助
