pickle与其他的持久化的比较

与marshal模块比较：marshal序列化的方法更加原始，其存在主要是支持python的 .pyc 文件。在序列化python对象时首选是pickle

与JSON模块比较：两者有本质不同，JSON是一个文本序列化协议，且在python和python外广泛应用；pickle是一个二进制序列化格式，仅用于python对象的序列化
